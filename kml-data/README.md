# KML Data Folder

This folder contains KML (Keyhole Markup Language) files for use with Google Earth and map visualizations on the Marco Polo Tech and Folklore site.

## About KML Files

KML files are XML-based geographic data files that can be imported into Google Earth, Google Maps, and other mapping applications.

## Files in this directory:

- `marco-polo-routes.kml` - Geographic data for Marco Polo trade routes and locations

## How to use:

1. Download any `.kml` file from this folder
2. Open it in Google Earth or upload it to Google Maps
3. Alternatively, link to these files directly from your map visualizations

## Creating new KML files:

You can generate KML files using:
- Google Earth Pro (export as KML)
- Online KML generators
- Manual XML creation following the KML 2.2 specification
